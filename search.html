Search Component Features

  • Click to focus Clicking the search icon automatically focuses the input field
  • Clear button An X icon appears when typing to clear the input field
  • Live results Search results appear as you type (after 2 characters)
  • Product navigation Click on any product to navigate to its page
  • Enter to search Press Enter to view all search results
  • Toggle search Click the search icon to open/close the search bar (mobile)
